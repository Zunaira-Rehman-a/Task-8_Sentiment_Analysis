# Task-8_Sentiment_Analysis
## Project Overview
This project focuses on sentiment analysis of customer reviews using transformer-based NLP models. The goal is to classify reviews into Positive, Negative, and Neutral sentiments.
## Model Used
- Pretrained Transformer Model (CardiffNLP RoBERTa / DistilBERT)
- HuggingFace Transformers Pipeline
## Features
- Sentiment classification (Positive, Negative, Neutral)
- Batch processing for efficient predictions
- Confidence score for each prediction
- Aspect-Based Sentiment Analysis (price, delivery, quality, service)
- Data visualization (bar chart & pie chart)
## Evaluation Metrics
- Accuracy
- F1-score
## Results
- Accuracy: ~0.77
- F1-score: ~0.87
## Files in Repository
- Task-8_Sentiment_Analysis.ipynb → Main notebook
- final_output.csv → Model predictions output
- requirements.txt → Required dependencies
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open notebook in Google Colab or Jupyter Notebook
3. Run all cells sequentially
